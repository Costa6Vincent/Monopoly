Welcome to Monopoly War!

Monopoly War takes all the main elements of monopoly and adds the function of armies. With armies, instead of having to pay rent you can simply taker over one's settlement(as long as your army is strong enough). This game was specially produced by: Vincent Costa, Tyler Cook, and Jessica Jacinto.

Pre-Alpha

This game is still in pre-alpha and is still being worked on. Many features as of right now are missing and some bugs are present within the code
